> A Python CLI tool for generating CLAUDE.md files for Cline AI projects.

## Features

- Generate CLAUDE.md project context files
- Support for multiple project types
- Customizable prompts
- Batch generation
- Python API for integration

## Installation

### From PyPI

```bash
pip install claude-prompt-gen-python
```

### From Source

```bash
git clone https://github.com/mizoz/claude-prompt-gen-python.git
cd claude-prompt-gen-python
pip install -e .
```

## Usage

### Command Line

```bash
# Generate CLAUDE.md
claude-prompt-gen

# Generate with custom project type
claude-prompt-gen --type webapp

# Generate with custom output
claude-prompt-gen --output custom-claude.md
```
